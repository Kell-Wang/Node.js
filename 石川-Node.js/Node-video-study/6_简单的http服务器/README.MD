## 用户注册，登录
> 接口
- 注册:
    + /user?act=register&user=aaa&pass=123456
    + {'ok':false, 'msg': '原因'}
- 登录:
    + /user?act=login&user=aaa&pass=123456
    + {'ok':false, 'msg': '原因'}

    + http://localhost:8020/1.html
    + http://localhost:8020/a.js
    + http://localhost:8020/button.jpg
- 请求接口
    + http://localhost:8020/user?act=xxx...
